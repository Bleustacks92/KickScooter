
Файлы прошивок:

0x1A = Первая плата процессор: GD32F130C8T6
0x1B = Первая плата процессор: GD32F130C6T6
0x2A = Вторая плата процессор: GD32F130C8T6
0x2B = Вторая плата процессор: GD32F130C6T6
0x3A = Третья плата процессор: GD32F130C8T6
0x3B = Третья плата процессор: GD32F130C6T6

Расшифровка названия файлов прошивок:

0 x 1 A

[0 x] 	= Hex файл
1		= Номер платы
A		= Тип процессора

Возможные виды процессоров:

A		= GD32F130C8T6
B		= GD32F130C6T6

Нумерацию и переделку плат можно по смотреть в папке - Платы.
